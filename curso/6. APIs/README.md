# 🌐 Consumo de APIs en Android (Retrofit + Kotlin) - 2º DAM

## 📌 Descripción
Este repositorio contiene el material del bloque de **consumo de APIs REST en Android** dentro del módulo de Programación Multimedia y Dispositivos Móviles (CFGS DAM).

El objetivo es que el alumnado aprenda a integrar datos externos en aplicaciones Android modernas utilizando Kotlin, coroutines y una arquitectura básica tipo MVVM.

---

## 🎯 Objetivos del bloque

Al finalizar este bloque, el alumnado será capaz de:

- Comprender qué es una API REST
- Realizar peticiones HTTP desde Android
- Consumir datos JSON en Kotlin
- Utilizar Retrofit para comunicación con servidores
- Manejar estados de carga, éxito y error
- Integrar llamadas asíncronas con coroutines
- Mostrar datos de API en Jetpack Compose

---

- Ejemplos guiados paso a paso
- Consumo progresivo de APIs simples a complejas

## 📌 Recomendaciones

- Probar APIs primero fuera de Android si es posible
- Entender bien los estados (loading/error)
- No mezclar lógica de red con UI
- Usar logs para depuración
