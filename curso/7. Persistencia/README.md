# 💾 Persistencia de datos en Android (Room + DataStore) - 2º DAM

## 📌 Descripción
Este repositorio contiene el material del bloque de **persistencia de datos en Android** dentro del módulo de Programación Multimedia y Dispositivos Móviles (CFGS DAM).

El objetivo es que el alumnado aprenda a almacenar datos de forma local en dispositivos Android utilizando bases de datos SQLite modernas con **Room** y almacenamiento ligero con **DataStore**.

---

## 🎯 Objetivos del bloque

Al finalizar este bloque, el alumnado será capaz de:

- Comprender la persistencia de datos en Android
- Utilizar Room como base de datos local
- Crear entidades, DAO y bases de datos
- Realizar operaciones CRUD
- Utilizar DataStore para preferencias simples
- Integrar persistencia con ViewModel
- Mostrar datos almacenados en Jetpack Compose
