# 🏗️ Arquitectura de Apps Android (MVVM básico) - 2º DAM

## 📌 Descripción
Este repositorio contiene el material del bloque de **Arquitectura de aplicaciones Android** dentro del módulo de Programación Multimedia y Dispositivos Móviles (CFGS DAM).

El objetivo es que el alumnado aprenda a estructurar aplicaciones Android de forma mantenible utilizando un enfoque moderno basado en **MVVM (Model - View - ViewModel)** con Kotlin y Jetpack Compose.

---

## 🎯 Objetivos del bloque

Al finalizar este bloque, el alumnado será capaz de:

- Comprender la arquitectura MVVM
- Separar lógica de UI y lógica de negocio
- Utilizar ViewModel correctamente
- Gestionar estado de forma estructurada
- Organizar un proyecto Android en capas
- Mejorar la escalabilidad del código
- Integrar arquitectura con Jetpack Compose

---

## 🧰 Herramientas

- Android Studio  
- Kotlin  
- Jetpack Compose  
- ViewModel (Android Architecture Components)  

---

## 🗂️ Contenidos

### 1. Introducción a la arquitectura

### 4. ViewModel
- Ciclo de vida

### 5. Flujo de datos
- Unidireccional (UDF)
- Comunicación View ↔ ViewModel
- Estado centralizado

### 6. Organización del proyecto
- Estructura por capas
- Separación de paquetes
- Buenas prácticas

- Primero hacer funcionar la app, luego estructurarla
- No mezclar UI con lógica de negocio
- Mantener ViewModels simples
- Pensar siempre en escalabilidad
