# 🧠 Curso: Kotlin Básico (2º DAM)

## 📌 Descripción

Este repositorio contiene el material del bloque de **Kotlin básico** dentro del módulo de Programación Multimedia y Dispositivos Móviles (CFGS DAM).

El objetivo es que el alumnado adquiera una base sólida del lenguaje Kotlin antes de entrar en desarrollo Android.

---

## 🎯 Objetivos del bloque

Al finalizar este bloque, el alumnado será capaz de:

* Comprender la sintaxis básica de Kotlin
* Utilizar variables, tipos de datos y operadores
* Crear funciones reutilizables
* Trabajar con estructuras de control (if, when, bucles)
* Utilizar listas y colecciones
* Entender clases y data classes
* Aplicar null safety correctamente

---
