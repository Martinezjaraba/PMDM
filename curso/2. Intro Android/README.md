# 📱 Introducción a Android con Kotlin (2º DAM)

## 📌 Descripción

Este repositorio contiene el material del bloque de **Introducción a Android** dentro del módulo de Programación Multimedia y Dispositivos Móviles (CFGS DAM).

El objetivo es que el alumnado comprenda la estructura básica de una aplicación Android moderna y sea capaz de crear sus primeras interfaces con Kotlin y Jetpack Compose.

---

## 🎯 Objetivos del bloque

Al finalizar este bloque, el alumnado será capaz de:

* Comprender la estructura de un proyecto Android
* Ejecutar aplicaciones en emulador o dispositivo físico
* Entender el ciclo de vida básico de una Activity
* Crear interfaces simples con Jetpack Compose
* Manejar eventos básicos de usuario
* Trabajar con estado simple en la UI

---

## 🗂️ Contenidos

### 1. Entorno de desarrollo

* Instalación y configuración de Android Studio
* Creación de proyectos
* Estructura de carpetas

### 2. Primeros conceptos de Android

* Qué es una Activity
* Ciclo de vida básico
* Archivo AndroidManifest

### 3. Introducción a Jetpack Compose

* Qué es Compose
* @Composable
* setContent

### 4. UI básica

* Text
* Button
* Column y Row
* Modifier

### 5. Estado en Compose

* remember
* mutableStateOf
* recomposición

### 6. Eventos de usuario

* onClick
* interacción básica
