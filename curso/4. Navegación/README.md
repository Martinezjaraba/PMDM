# 🔀 Navegación en Android con Jetpack Compose (2º DAM)

## 📌 Descripción
Este repositorio contiene el material del bloque de **Navegación en aplicaciones Android** dentro del módulo de Programación Multimedia y Dispositivos Móviles (CFGS DAM).

El objetivo es que el alumnado aprenda a estructurar aplicaciones con múltiples pantallas utilizando **Jetpack Navigation Compose**, gestionando el paso de datos entre vistas y la organización de la UI.

---

## 🎯 Objetivos del bloque

Al finalizar este bloque, el alumnado será capaz de:

- Crear aplicaciones con múltiples pantallas
- Utilizar Navigation Compose
- Definir rutas de navegación
- Pasar parámetros entre pantallas
- Gestionar flujos básicos de navegación
- Integrar navegación con estado y ViewModel
- Diseñar estructuras de app escalables

---

## 🧰 Herramientas

- Android Studio  
- Kotlin  
- Jetpack Compose  
- Navigation Compose
