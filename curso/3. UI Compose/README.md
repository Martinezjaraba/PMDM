# 🎨 UI con Jetpack Compose (2º DAM)

## 📌 Descripción
Este repositorio contiene el material del bloque de **Interfaz de Usuario con Jetpack Compose** dentro del módulo de Programación Multimedia y Dispositivos Móviles (CFGS DAM).

El objetivo es que el alumnado aprenda a construir interfaces modernas en Android utilizando Kotlin y el sistema declarativo **Jetpack Compose**.

---

## 🎯 Objetivos del bloque

Al finalizar este bloque, el alumnado será capaz de:

- Crear interfaces de usuario con Jetpack Compose
- Utilizar componentes básicos de UI (Text, Button, TextField, etc.)
- Organizar layouts con Column, Row y Box
- Manejar estado en Compose
- Entender la recomposición
- Aplicar estilos básicos a la UI
- Crear interfaces interactivas

---

## 🧰 Herramientas

- Android Studio

- Jetpack Compose
